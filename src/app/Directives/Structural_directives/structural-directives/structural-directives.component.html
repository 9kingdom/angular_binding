Structural directives are responsible for HTML layout. They shape or reshape the DOM's structure, typically by adding, removing, or manipulating elements.

The most common attribute directives are as follows:

  • NgIf—adds and removes a set of CSS classes.
  • NgFor—adds and removes a set of HTML styles.
  • NgSwitch—adds two-way data binding to an HTML form element.

You can apply many attribute directives to one host element. You can only apply one structural directive to a host element.
