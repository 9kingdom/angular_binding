An Angular custom (attribute) directive can be simply described as a component without a template.

Custom directives are generally special case of attribute directive. They are used when no additional template is needed. The directive can execute logic and apply visual changes to the element it is applied to. This is useful if you want to alter the behavior or style of existing HTML-elements, without wrapping them into a new component.

Template:

Creating a custom directive is easy. Just create a new class and decorate it with the @Directive decorator:
ng generate directive example
*make sure that the directive is declared in the app-module.

Using the directive selector:

Every directive has a selector, which is used to apply the directive to elements in your templates.
The directive can be applied to HTML-elements in your templates using its selector.
