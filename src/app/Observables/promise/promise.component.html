Promise is used mostly with HTTP requests. A Promise constructor takes two parameters: resolve and reject.

The resolve() function is invoked when the asynchronous task is completed and renders the result of the assigned tasks.

The reject() function is invoked when the assigned task fails and returns predefined error.

Promise has three probable states:

  • Pending: Initial state, neither fulfilled nor rejected
  • Fulfilled: onFulfilled() will be invoked, corresponds to resolve()
  • Rejected: onRejected() will be invoked, corresponds to reject()
